The software cellPACK enables researchers to assemble mesoscale models of entire cells, organelles or viruses. It's user interface is currently limited to a plug-in that runs in Maya or Cinema 4D, which are professional 3D softwares with steep learning curves. To make cellPACK accessible to a broader user base, Graham Johnson and Sam Hertig created a web interface for interactively visualizing cellPACK data, using JavaScript, jQuery, Three.js and assembled models of HIV components. The hierarchical tree is created dynamically by parsing a Collada-file of the structural model.
