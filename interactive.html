Interactive

We design interactive visualizations that enable you to present your scientific or technical data to your customers and audience in an immersive and engaging process.

For the Zurich-based software company Comerge, we help develop a web-based 3D-visualization app to render and simulate warehouses in real-time using Javascript and the WebGL library Three.js.

This prototype of an interactive visualization tool was created to explore structural biological data in a web browser. It was designed as the front-end for cellPACK, which was developed by Graham Johnson et al.

We can simplify your data analysis and visualization pipeline using Python and SQL. Multidomain Assembler (MDA) is an algorithm for modeling and visualizing large multidomain proteins, fully embedded in the molecular graphics software UCSF Chimera.
